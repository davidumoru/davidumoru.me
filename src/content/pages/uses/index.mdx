## This site

Everything here is written up in more detail on [Colophon](/colophon). In short:

- [Astro](https://astro.build/) for the framework
- MDX for pages and posts
- [Radix Colors](https://www.radix-ui.com/colors) + custom CSS variables
- [Astro DB](https://docs.astro.build/en/guides/astro-db/) and [Turso](https://turso.tech/) for the guestbook
- Hosted on [Vercel](https://vercel.com/)

Typography: **Editorial New** for headings, **Inter** for body text.
